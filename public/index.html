קשרים משפחתיים
📖 מדריך למנהל
🏁 התחלה מהירה
1. הוסף את בני המשפחה — לחץ על לשונית "משפחה" ← "+ הוסף משפחה"
2. הגדר ערוץ תקשורת — לכל משפחה ניתן לשמור מספר WhatsApp ואימייל
3. הוסף חגים — מדף "ראשי" ← "+ הוסף אירוע משפחתי". שבתות נוצרות אוטומטית
4. הזמן חדרים — לחץ על חדר בלשונית "חדרים", בחר שבת/חג ומשפחה
5. רשום סעודות — לשונית "סעודות" ← "+ הוסף"
6. שלח דוח לאמא — לשונית "התראות" ← "שלח דוח לאמא"
🔐 ניהול הרשאות
• מנהל — לחץ על שמך בפינה הימנית העליונה ← "הפוך למנהל"
• נעילת חדר — רק מנהל יכול לנעול חדר (מונע הזמנות נוספות)
• הרשאות Firestore — כל משתמש מחובר יכול לקרוא ולכתוב. ניתן להגביל ב-firestore.rules
📅 יזכור ומועדים
• ימי יזכור מוצגים בלוח השנה עם מסגרת אדומה
• ימי יזכור בישראל: יום כיפור, שמיני עצרת, שביעי של פסח, שבועות
• הנתונים נטענים מ-Hebcal.com אוטומטית
🛠 תחזוקה שוטפת
• גיבוי — הנתונים שמורים ב-Firebase Firestore. ניתן לייצא מ-Firebase Console
• עדכון פרשות — נטען אוטומטית מ-Hebcal ל-91 ימים קדימה
• ניקוי נתוני בדיקה — Firebase Console → Firestore → מחק מסמכים ידנית
• Deploy — firebase deploy מהתיקייה הראשית
• כתובת האפליקציה — steinberg-family-c0c32.web.app
